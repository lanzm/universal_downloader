本程序是一款基于 Python 内置 Web 服务的无依赖、高颜值本地音视频提取器。

操作指引：
1. 搜索提取：在“搜索提取”标签页中，输入任意关键字，点击执行即可获取解析结果。点击每行最右侧的 提取音频 按钮即可触发下载。
2. 链接直提：直接在搜索框粘贴 B站视频链接、BV号或网易云歌曲链接，回车或点击执行，系统会跳过搜索直接拉起下载。
3. 下载自动监控：无论是点击表格下载，还是粘贴链接回车，系统都会自动切入到“下载日志”页面并实时呈现滚动的后台转码进度，无需手动查看。
4. 修改参数：在“配置参数”中可以一键浏览系统目录来修改音频保存目的地或输出质量。
This program is a dependency-free, high-visual-quality local audio/video extractor powered by Python's built-in HTTP server.

User Guide:
1. Search & Extract: In the "Search & Extract" tab, enter any keyword and click Execute to search. Click the Extract Audio button on the far right of any row to download.
2. Direct Extract: Paste a Bilibili link, BV code, or NetEase song link directly into the search input and press Enter to extract without searching.
3. Auto Logs Monitor: Triggering any download will automatically switch the view to the "Download Logs" page, rendering real-time scrolling console details.
4. Configuration: Browse your local directories in "Settings" to easily modify the download folder, output formats, visual themes, and languages.
